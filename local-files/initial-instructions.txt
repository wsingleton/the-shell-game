	Attached to this message is your pem key file that you will use to connect to the EC2 instance. Download it and use `ssh -i <pem-file> <username>@<ec2-ip-or-dns>` to access the instance (your username is the same as the pem file name). Once connected you will find a directory named `the-shell-game`, enter it and read the README inside using `cat`. What follows are a series of rooms with simple tasks that aim to get the user more comfortable with working on the command line.
